HTML Concepts

HTML Tags

HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content. HTML tags contain three main parts: opening tag, content and closing tag. But some HTML tags are unclosed tags. These tags are also called self-closing tags.
When a web browser reads an HTML document, browser reads it from top to bottom and left to right. HTML tags are used to create HTML documents and render their properties. Each HTML tags have different properties. An HTML file must have some essential tags so that web browser can differentiate between a simple text and HTML text. You can use as many tags you want as per your code requirement. All HTML tags must enclosed within opening and closing conical these brackets. Every tag in HTML is used to perform different tasks. If you have used an open tag <>, then you must use a close tag </tag> (except some tags)

Here are different types of HTML Tags and some examples :

  1. Unclosed HTML Tags :

    • <br> - used to give a break between lines, enter a new line
    • <hr> - stands for horizontal Rule, used to add a line across the webpage

  2. HTML Meta Tags

    • DOCTYPE
    • title
    • link
    • meta
    • style

  3. HTML Text Tags

    • <p> - paragraph tag
    • <h1> - heading tag
    • <strong> - used to make text look bold
    • <code> - used to show code in html document

  4. Link Tags

    • <a> - anchor tag, used to display hyperlinks to different webpages or local navigation
    • <base> - used to specify a base url for all url's in a page, a document can have only one base tag

  5. List Tags

    • <ul> - used to represent an unordered list, list without numbering or order
    • <ol> - used to represent an ordered list, list with numbering
    • <l> - used to represent singular elements of an ordered or unordered list

Void Elements

A void element is an element whose content model never allows it to have contents under any circumstances. Void elements can have attributes. They only have one opening tag and no closing tag, they can have different attributes but no content

Examples of such elements are :

  • <img> - img element
  • <meta> - meta element
  • <input> - input element

A semantic element clearly describes its meaning to both the browser and the developer.

Examples of Non-semantic elements are : <div> & <span> , they tell nothing about their content

Examples of Semantic elements are : <form> & <table> , they tell information about their content

Section-Element

The <section> element defines a section in a document. A section is a thematic grouping of content, typically with a heading.
Examples of where a <section> element can be used:

  • Chapters
  • Introduction
  • News items
  • Contact information
A web page could normally be split into sections for introduction, content, and contact information. This portion in the html has been written in the section element.

Article-Element

The <article> element specifies independent, self-contained content. An article should make sense on its own, and it should be possible to distribute it independently from the rest of the web site. Examples of where the <article> element can be used:

  • Forum posts
  • Blog posts
  • User comments
  • Newspaper articles
A web page could normally be split into sections for introduction, content, and contact information. This portion in the html has been written in the article element.

Header Element

The <header> element represents a container for introductory content or a set of navigational links. A <header> element typically contains:

  • one or more heading elements
  • logo or icon
  • authorship information

Footer Element

The <footer> element defines a footer for a document or section. A <footer> element typically contains:

  • authorship information
  • copyright information
  • contact information

Meta Tags

The <meta> tag defines metadata about an HTML document. Metadata is basically data or Information about other data. These tags usually go inside the head element and are not visible on the browser. They are used to specify things like the character set used in the webpage, the keywords, the viewport , etc.
Metadata is used by browsers, search-engines and other web services. Meta tags are self-closing tags
